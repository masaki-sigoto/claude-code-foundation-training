# NotebookLM コピペ用プロンプト集（優先10点）

このファイルは、NotebookLM の「Studio」→「インフォグラフィック」入力欄に **そのままコピペで使える** プロンプト集です。優先度順に並べてあります。

## 使い方

1. NotebookLM のノートブックを開く
2. 右側「Studio」パネル → 「インフォグラフィック」をクリック
3. プロンプト入力欄に、下記のいずれかのブロックを **見出し以外** まるごと貼り付け
4. 「生成」を押す
5. 完成したらタイトルを `IMG-012` のような ID で命名し直す（後でリポジトリ取り込み時に管理しやすくなる）

注意：複数同時に依頼すると NotebookLM 側で1つにまとめられる挙動があるため、**1点完成 → 次の1点** を1点ずつ進めてください。

## 1. IMG-012：指示文クッション方式の全体像（第3章・最重要）

```
タイトル：指示文クッション方式の全体像

非エンジニアの業務スタッフ向けに、以下の流れを横長のフローチャートで分かりやすく可視化してください。

【描く流れ】
1. 人（雑な要望をぼやくイメージ）
2. → 通常チャットAI（ChatGPT / Claude.ai / Gemini）が「整った指示文」に整形（クッション役）
3. → Claude Code が実ファイルを操作して作業
4. → Claude Code から「途中報告（A案・B案どちらにしますか）」が出る
5. → その報告を再びチャットAIに貼って壁打ち（戻りループは点線・柔らかく）
6. → 整理された「次の指示」を Claude Code に戻す

【たとえ】上司との往復書簡。引っ越し業者に「いい感じに運んで」ではなく荷物リストと搬入先を渡す。

【スタイル】横長フロー、戻りループは点線、親しみやすいフラットイラスト、日本語キャプション、文字最小限、アイコン中心、矢印は交差させない。

【避けたい】技術的なシステム図、コード、英語の専門用語、黒い画面、ロボット感の強いアイコン。
```
