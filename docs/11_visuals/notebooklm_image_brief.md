# NotebookLM画像案依頼メモ

このドキュメントは、NotebookLMに画像案を相談するときに使えるよう、各画像の意図を簡潔にまとめたものです。
詳細仕様は `image_insertion_plan.md` を参照してください。本ファイルは「依頼文として貼りやすい簡潔メモ」に特化しています。

---

## 使い方

1. NotebookLMに教材一式（GitHubリポジトリ、もしくは `docs/` 配下）をソースとして登録
2. 本ファイルから1つ画像を選び、依頼メモをコピー
3. 「この内容で初心者向け画像案を3パターン提案してください」とNotebookLMに投げる
4. 出てきた案を比較し、採用案を IMG-XXX に紐付けて記録する
5. 採用案は `image_insertion_plan.md` の該当 IMG エントリに「採用版メモ」として追記

---

## 共通の前提（全画像に共通）

依頼の冒頭に以下を毎回貼ると、トーンが揃いやすいです。

```
前提：
- 対象は非エンジニアの業務スタッフ（初心者）
- トーンは落ち着いた信頼感（おしゃれより明瞭）
- 配色は indigo 系と相性の良い柔らかいトーン
- 日本語キャプション中心、英語はサービス名のみ可
- 黒い画面・コード断片・威圧的なロボットは避ける
- 1画像1メッセージを徹底、情報を詰め込まない
- 絵文字は使わない
```
