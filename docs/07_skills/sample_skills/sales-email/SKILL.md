# 営業メール作成スキル

## このスキルは何をするか

営業活動で繰り返し発生する定型メール（初回アプローチ、アポ後フォローアップ、提案後の追いかけ、失注後の関係維持など）を、相手の状況と目的に合わせて作成します。ゼロから書くのではなく、本スキルのパターンと手順に沿って組み立てることで、表現ぶれや失礼な言い回しを防ぎます。

## いつ使うか

- 新規顧客への初回アプローチメールを作成するとき
- 商談・打ち合わせ後のお礼とネクストアクション連絡を送るとき
- 提案書送付後、一定期間返信がない相手にフォローアップするとき
- 失注が確定した後、関係を維持するためのクロージングメールを送るとき
- 既存のメール文面を「より丁寧に」「より簡潔に」など方向性を指定して直すとき

## 使ってはいけない場面

- 契約書・同意書など法的効力のある文書の作成（別途法務確認が必要）
- クレーム対応・謝罪メール（状況を誤解したまま文面化するリスクが高いため、個別に慎重対応）
- 相手が個人情報・機微情報を含む返信をしている案件（生の情報を AI に渡す前に社内ルールを確認）

## 事前に準備するもの

- 相手情報：会社名、部署、役職、担当者氏名、関係性（初回／既存／失注後など）
- 目的：このメールで相手に何をしてほしいか（返信／日程調整／資料確認／再検討）
- 商材：紹介する商品・サービス名と、相手にとっての価値を一言
- 前回のやり取り（あれば）：直近のメール本文や打ち合わせ要旨
- 署名・連絡先情報

## 手順

1. **メール種別を特定する**
   「初回アプローチ」「お礼＋次アクション」「フォローアップ」「失注後」のどれに該当するかを確認する。不明な場合はユーザーに質問する。
2. **相手属性を整理する**
   会社規模・業界・役職から、フォーマルさのレベル（堅め／標準／ややカジュアル）を決める。
3. **件名を作る**
   件名は 25 文字以内を目安に、用件が一目で分かる形にする。「ご提案」「ご確認」「日程」など目的を冒頭に置く。
4. **本文の骨格を組む**
   - 宛名（会社名＋部署＋役職＋氏名＋「様」）
   - 挨拶（初回：名乗り／2 回目以降：お世話になっております）
   - 用件の要旨（3 行以内）
   - 詳細・背景
   - 相手に取ってほしい行動（返信期限・選択肢）
   - 締め・署名
5. **トーンを調整する**
   押しが強すぎないか、逆にへりくだりすぎていないかを確認。目安は「初対面の取引先に直接話しかけるときの口調」。
6. **最終チェック（次項の品質チェック観点に沿う）**
7. **ユーザーに確認を取る**
   送信前にユーザーが内容を確認する前提で提示する。

## 成果物の形式

```
件名: （ここに件名）

（ここに本文）

-----
（署名）
```

本文と件名は必ず分けて提示する。複数パターン欲しいと依頼された場合は「パターン A／パターン B」のように並べる。

## 品質チェック観点

- [ ] 宛名の会社名・氏名・敬称に誤りがない
- [ ] 件名だけで用件が分かる
- [ ] 冒頭 3 行で目的が伝わる
- [ ] 相手に求める行動（返信・日程調整など）が明確
- [ ] 敬語が過剰／不足していない（二重敬語、「させていただく」の乱用を避ける）
- [ ] 商材の押し売り感が強すぎない
- [ ] 誤字脱字がない
- [ ] 日付・金額・数値は具体的かつ正確
- [ ] 署名がある

## よくある失敗

- **用件が分からない件名**：「ご連絡」「お世話になっております」だけの件名は埋もれる。→ 必ず目的を入れる。
- **相手メリットがなく、こちら都合だけ書いている**：「弊社の〜をご紹介したく」だけでは読まれない。→ 「御社の◯◯という課題に対して」と相手起点で書く。
- **長すぎる**：1 通で全部伝えようとしない。→ 初回は「会話のきっかけ」を作るだけでよい。
- **フォローアップで督促感が強すぎる**：「ご返信お待ちしております」を連発しない。→ 「別件でご参考になりそうな情報がありましたので」など自然な接点を作る。

## 使用例

### 入力例

> 来週、株式会社サンプル物産の採用担当・山田様（課長）に、弊社の採用管理システム「HireBoost」を紹介したい。初回アプローチのメールを書いてほしい。山田様とは面識なし。先方は人事部で、中途採用の工数増大に悩んでいると業界記事に出ていた。

### 出力例

```
件名: 中途採用工数に関するご参考情報のご送付

株式会社サンプル物産
人事部 課長
山田様

突然のご連絡、失礼いたします。
株式会社◯◯の△△と申します。

貴社が中途採用の強化に取り組まれている旨を業界誌で拝見し、
同じ課題に向き合う企業様をご支援してまいった経験から、
ご参考になりそうな情報がございましたのでご連絡いたしました。

弊社では、中途採用の初期対応工数を削減する採用管理システム
「HireBoost」を提供しております。
導入企業様では、書類選考〜一次面接調整までの工数を
平均で約 40% 削減した事例がございます。

もしご興味をお持ちいただけましたら、
15 分ほどのオンラインでのご説明の機会を頂戴できれば幸いです。
ご都合のよい候補日を 2〜3 ついただけますでしょうか。

ご多忙のところ恐縮ですが、ご検討のほどよろしくお願い申し上げます。

-----
株式会社◯◯
△△ △△
Mail: xxx@example.com
Tel: 03-0000-0000
```
