# Claude Code 基礎勉強会 スライド構成案（全45枚）

対象：非エンジニアの業務スタッフ（初心者向け）
想定時間：90分版（演習1〜2個）／120分版（演習3個）／半日版（演習5個）
方針：すべての重要概念に「日常のたとえ」「仕事のたとえ」を必ず添える。詳細は講師補足へ逃がす。

## 教育フレームワーク（各重要概念で守る6ステップ）
1. まず一言で言うと
2. 日常でたとえると
3. 仕事でたとえると
4. Claude Codeではこうなる
5. よくある勘違い
6. 最初に覚えるべき使い方

## 統一比喩（揺らさない）
- Claude Code＝作業してくれるAIエージェント／机に来て作業してくれるアシスタント
- CLAUDE.md＝プロジェクトの取扱説明書／家事代行さんに渡す家のルール
- memory＝作業しながら覚えた注意点
- skills＝繰り返し作業の専用レシピ／料理のレシピ
- MCP＝外部ツールとの接続口／外の倉庫・別の部屋への鍵
- hooks＝決まったタイミングで動く自動スイッチ／玄関を出る前のチェック
- subagents＝専門作業を任せる別担当AI／引っ越しの荷造り担当
